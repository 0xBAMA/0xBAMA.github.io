Knights Tours via Recursive Backtracing

  This project came up when I found some of my old code from CS3610, Data Structures. I had written it very poorly, hardcoded the size of the board, and certain starting positions would create issues - I took this chance to fix these problems and make it work for an arbitrary sized board. A 'Knight's Tour' is a pattern in which a knight can visit all the squares on a chessboard - they exist for all square chessboards which have an edge length greater than 4 - I found a pretty good discussion of this fact here.
